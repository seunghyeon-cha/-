# GitHub Pages 배포 완료!

## 배포 상태

gh-pages 브랜치가 성공적으로 GitHub에 푸시되었습니다!

## 마지막 단계: GitHub Pages 활성화

다음 단계를 통해 GitHub Pages를 활성화해주세요:

1. GitHub 저장소 페이지로 이동: https://github.com/seunghyeon-cha/-

2. Settings 탭 클릭

3. 왼쪽 메뉴에서 "Pages" 클릭

4. "Build and deployment" 섹션에서:
   - Source: "Deploy from a branch" 선택
   - Branch: "gh-pages" 선택
   - Folder: "/ (root)" 선택

5. Save 버튼 클릭

6. 약 1-2분 후, 다음 URL에서 웹사이트 확인:
   https://seunghyeon-cha.github.io/-/
