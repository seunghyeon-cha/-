## 📊 현재 상태 분석

### 기존 메인 페이지 구조
```
1. 히어로 섹션
   - 텍스트 중심 (제목, 설명)
   - CTA 버튼 2개

2. 카테고리 카드 (3개)
   - 이모지 아이콘만 사용 🏞️ 🍜 🏨
   - 실제 이미지 없음

3. 개발 현황 박스
   - 텍스트 정보만
```

### 문제점
- ✗ **시각적 임팩트 부족**: 이미지가 없어 단조로움
- ✗ **정보 전달력 약함**: 텍스트만으로는 매력 전달 어려움
- ✗ **사용자 참여 유도 부족**: 클릭 유도 요소 부족
- ✗ **bobaedream 스타일과 거리감**: 이미지 중심 레이아웃 필요

## 🎯 개선 목표

### 벤치마크: bobaedream.co.kr 스타일
bobaedream은 다음과 같은 특징을 가집니다:
1. **메인 배너 슬라이더**: 큰 이미지로 시선 집중
2. **카테고리 이미지 카드**: 실제 사진으로 정보 전달
3. **컨텐츠 그리드**: 다양한 게시물을 이미지와 함께 표시
4. **비주얼 중심**: 텍스트보다 이미지 비중 높음

### 예림투어 적용 방향
- ✓ **여행 감성 극대화**: 아름다운 여행지 사진 활용
- ✓ **카테고리별 대표 이미지**: 관광지/맛집/숙소 실제 사진
- ✓ **추천 컨텐츠**: 인기 여행지/맛집 이미지 그리드
- ✓ **반응형 디자인**: 모바일에서도 이미지 최적화

## 🖼️ 추가할 이미지 섹션

### 1. 히어로 섹션 (메인 배너 슬라이더)
**위치**: 페이지 최상단
**목적**: 첫 방문 시 강렬한 인상 제공

**구성**:
```
┌─────────────────────────────────────────┐
│  [←]  전국 최고의 여행지를 만나보세요  [→]  │
│                                         │
│       대형 배경 이미지                     │
│       (제주도, 부산, 강원도 등)              │
│                                         │
│         [여행 시작하기 버튼]                │
│                                         │
│       ● ○ ○ ○  (슬라이드 인디케이터)        │
└─────────────────────────────────────────┘
```

**세부 스펙**:
- 슬라이드 개수: 4-5개
- 자동 전환: 5초 간격
- 이미지 크기: 1920x600px (데스크탑), 768x400px (모바일)
- 배경 이미지 예시:
  1. 제주도 성산일출봉
  2. 부산 해운대
  3. 강원도 설악산
  4. 경주 불국사
  5. 전주 한옥마을

**기술 스펙**:
```tsx
- 자동 슬라이드 + 수동 네비게이션
- 터치 스와이프 지원 (모바일)
- Lazy loading으로 성능 최적화
- ARIA 접근성 (role="region", aria-label)
```

```tsx
interface HeroSlide {

// 자동 전환: 5초
// 페이드 전환 애니메이션
// 키보드 네비게이션 지원 (←, →)

```

### 반응형 브레이크포인트
```css
- Mobile: < 640px (sm)
- Tablet: 640px - 1024px (sm ~ md)
- Desktop: > 1024px (lg)

이미지 크기:
- Desktop: 1920x600px
- Tablet: 1024x500px
- Mobile: 640x400px
```

### 성능 최적화
```
- Lazy Loading: 스크롤 시 이미지 로드

- Placeholder: blur 효과
- Priority: 히어로 이미지는 우선 로드
```
